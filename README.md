Map Edit Tool<a name="TOP"></a>
===================
---
> C++ / Qt5 项目

二维地图编辑器项目。能够在地图图片中创建路标点，创建道路，支持较少的批量编辑操作。

## 需求分析
- 支持导入地图背景图片
- 支持通过点击、拓拽的方式插入地图点、地图边
- 支持地图点、边的批量编辑
- 支持地图数据整体的打包导出和导入
